# Contributing

## Linting

### YAML

We use [yamllint](https://github.com/adrienverge/yamllint) and [yamlfix](https://github.com/lyz-code/yamlfix).

```console
uv tool install yamlfix
uv tool install yamllint
```

[uv](https://github.com/astral-sh/uv) is the modern Python package and project manager.

### Markdown

We lint Markdown files using [DavidAnson/markdownlint](https://github.com/DavidAnson/markdownlint).
